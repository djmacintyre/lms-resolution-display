[% "PLUGIN_RESOLUTION_DISPLAY_DESC" | string %]

Available title format tokens for use in Settings > Formatting > Title Format:

  • RESOLUTION — e.g. "24/96 FLAC", "DSD128", "320kbps MP3"
  • BITDEPTH — e.g. "24", "16"
  • SAMPLERATE_KHZ — e.g. "44.1", "96"
  • SOURCEFORMAT — e.g. "FLAC", "MP3"
  • DECODE_RESOLUTION — e.g. "24/96* FLAC" (bit-perfect), "24/192>96 FLAC" (downsampled)
  • DECODE_RES_SHORT — e.g. "96*", "192>96", "D128", "320k"
  • SIGNAL_PCT — e.g. "87%" (wireless signal strength, empty for wired)
  • SIGNAL_BARS — single-char bar: "▄" (glyph) or ":" (ASCII)
  • BUFFER_PCT — e.g. "95%" (decode buffer fullness)
  • HEALTH — "●" good / "◐" degraded / "○" bad (or OK / ! / !!)
  • HEALTH_LINE — e.g. "W87 B95" (signal + buffer, 7 chars)

Example: create a title format TITLE (RESOLUTION) and select it in your player's screensaver or Now Playing display settings.
